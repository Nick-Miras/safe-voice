# Models
This is the directory the contains the Random Forest model used for ASR.

## Data
The dataset used to train the model is the Common Voice dataset initiated by the Mozilla Foundation. "The Common Voice corpus is a massively-multilingual collection of transcribed speech intended for speech technology research and development" (Ardila et al., 2020). It is under a Creative Commons license.

## References
1. Ardila, R., Branson, M., Davis, K., Henretty, M., Kohler, M., Meyer, J., Morais, R., Saunders, L., Tyers, F. M., & Weber, G. (2020). Common Voice: A Massively-Multilingual Speech Corpus (arXiv:1912.06670). arXiv. https://doi.org/10.48550/arXiv.1912.06670
